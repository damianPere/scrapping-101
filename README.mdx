### Instalacion de modulos para la API


1- Ejecutar desde la raiz del proyecto en la terminal el siguiente comando  `pip install -r requirements.txt`

2- Ejecutar `pip install werkzeug`

3- Verificar que node este instalado ejecutando el siguiente comand `node -v`

4- Ejecutar el siguiente comando `npm i serverless -g`



### Instalacion de modulos para la web app


1- Desde la terminal ubicarse en `src/ui`

2- Verificar que node este instalado ejecutando el siguiente comand `node -v`

3- Ejecutar el siguiente comando `npm i`


### Correr API localmente

1- Ejecutar `serverless wsgi serve`

2- Desde el explorador dirigirse a `http://127.0.0.1:5000/scrape`


### Correr web app localmente

1- Desde la terminal ubicarse en `src/ui`

2- Ejecutar `npm run dev`

3- Desde el explorador dirigirse a `http://localhost:4173/
