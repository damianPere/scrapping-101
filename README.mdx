### Correr repositorio localmente


1- Ejecutar desde la raiz del proyecto en la terminal el siguiente comando  `pip install -r requirements.txt`
2- Instalar `pip install werkzeug``
3- ejecutar `serverless wsgi serve`
4- Desde el explorador dirigirse a `http://127.0.0.1:5000/scrape`
